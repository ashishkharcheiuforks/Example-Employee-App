# Example-Employee-App
A Simple Android example app.

## Key Features:

* Androoid Jetpack componets ---> 

     1. MVVM
     2. LiveData
     3. Pagination
     4. Data Binding
     5. Room
     6. WorkManager
     7. Lifecycles
     
* androidx library
     
* dagger2 for DI

* Retrofit for RESTful api data retrival

* GSON for retrofit data conversion

* android JUnit test

* Kotlin Coroutines

* R8 for code shriking
